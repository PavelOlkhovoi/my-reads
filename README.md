# MyReads. React project

## Overview
It is my first react project for self-education. It is Book Tracking App. Users can use the app to manage the reading process. You can categorize books for different shelves like "Reading", "Want to read", and "Read". You can also searching new books and add books to shelves.

### Usage
You can clone the project, install all its dependencies and run it with 'npm start' command.
